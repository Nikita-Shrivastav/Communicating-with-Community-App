## üìã Production Notes

### Technical Requirements
- **Duration:** 15-30 seconds (App Store requirement)
- **Resolution:** 1920x1080 minimum (or device-native resolution)
- **Frame Rate:** 30 fps
- **Format:** .mov, .m4v, or .mp4
- **Audio:** Optional but recommended for accessibility
- **Captions:** Strongly recommended for accessibility

### Video Style
- Clean, simple interface demonstrations
- Focus on large touch targets and clear visual feedback
- Show real-world usage scenarios
- Highlight accessibility features
- Demonstrate multi-language support
- Keep transitions smooth and not too fast

### Recording Tips
1. Use iOS Simulator or real device screen recording
2. Clean up any notifications or personal data
3. Use a generic Apple ID or demo account
4. Consider using a physical device for more authentic touch interactions
5. Record at 60fps and export at 30fps for smoother playback

## üé¨ Recording Tools

### iOS/iPadOS
1. **Built-in Screen Recording**
   - Settings > Control Center > Screen Recording
   - Swipe down Control Center, tap record button
   - 3-second countdown before recording starts

2. **QuickTime (with Mac)**
   - Connect device via USB
   - Open QuickTime > New Movie Recording
   - Select iPhone/iPad as camera source
   - Higher quality output

### macOS
1. **Screenshot.app (‚åò + Shift + 5)**
   - Built-in screen recording
   - Select area or full screen
   - Includes cursor in recording

2. **QuickTime Player**
   - File > New Screen Recording
   - Simple, native tool

### Professional Tools
- **ScreenFlow** (Mac) - Advanced editing, annotations
- **Camtasia** (Mac/Windows) - Professional screen recording
- **iMovie** (Mac/iOS) - Free editing and effects
- **Final Cut Pro** (Mac) - Professional video editing

## ‚úÇÔ∏è Editing Workflow

### Basic Editing Steps
1. **Import footage** into editing software
2. **Trim clips** to desired length (aim for 25-28 seconds of content)
3. **Add text overlays** with fade in/out
4. **Insert transitions** between scenes (0.3-0.5 second fades)
5. **Add background music** (optional, low volume)
6. **Include voiceover** (optional, clear audio)
7. **Add captions** for accessibility
8. **Color correct** if needed (maintain vibrant app colors)
9. **Export** in required format and resolution
10. **Review** on actual device before submission

### Export Settings (Recommended)
- **Format:** H.264 (most compatible)
- **Resolution:** 1920x1080 (Full HD)
- **Frame Rate:** 30 fps
- **Bitrate:** 15-20 Mbps (high quality)
- **Audio:** AAC, 256 kbps, 48 kHz
